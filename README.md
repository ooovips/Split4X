# Split4X

縦長の画像を4分割し、プレビュー、パン/ズーム、間隔調整機能付きで表示します。  
Split a tall image into 4 images with preview, pan/zoom, and gap controls.

## 使い方

1. 画像をクリックまたはドラッグで読み込みます。
2. ズーム/Gap/出力幅を調整します（Gap とは各画像の間の隙間です）。
3. ズームしている場合はドラッグで画像を移動できます。
4. `すべてダウンロード`をクリックしてダウンロードします。
5. `フレームを表示`のチェックを外すと見た目上のフレームが消えて原画が見られます（出力画像に影響は無し）
6. `Gap を有効`のチェックを外すと隙間のない4分割画像になります。

詳しくは [ヘルプ](https://ooovips.github.io/Split4X/help.html) をご覧ください。

# Usage

1. Load an image by click/drop.
2. Adjust pan/zoom/gap/output width (Gap refers to the space between each image).
3. When zoomed, drag to move the image.
4. Click `すべてダウンロード` and download.
5. Uncheck `レームを表示` to remove the visual frames and view the original image (output images remain unaffected).
6. Uncheck `Gap を有効` to create a seamless four-panel image.

For more details, see [Help](https://ooovips.github.io/Split4X/help.html).
